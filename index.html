O jogo consiste em três partes principais:

  • Colocando as peças: Esta é a fase inicial onde cada jogador coloca um peça alternadamente. Caso um dos jogadores forme uma linha horizontal ou vertical com três peças, ele terá o direito de remover uma peça de seu adversário.
  • Movendo as peças: Esta fase se inicia quando os jogadores colocarem suas nove peças. Deve mover sua peça para uma casa do lado. Caso um dos jogadores tenha somente 3 peças em jogo, ele pode movar a peça para qualquer casa vazia.
  • Removendo peças adversárias: Em qualquer uma das fases acima quando um jogador forma uma "trilha" com 3 peças, ele poderá remover uma peça de seu adversário, porém, não pode remover uma peça que faz parte de uma trilha.

    • O jogo termina em 3 situações:

  • Se um jogador reduzir as peças de seu adversário para 2.
  • Se um jogador deixar seu adversário sem nenhuma jogada válida.
  • Se ambos jogadores estiverem com 3 peças em jogo e, em 10 jogadas não houver vencedor.
